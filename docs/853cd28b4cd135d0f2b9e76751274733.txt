1. Overview

Smart Bharat is an AI-powered civic platform that enables citizens to discover government services, apply for schemes, report civic issues, manage documents, and track applications from a single unified dashboard.

This document focuses on the Information Architecture (IA) and User Navigation Flows for the web application.

9. Design Principles

Dashboard-first navigation to centralize user activity.

Maximum of 2вЂ“4 clicks to complete primary tasks.

Simple, shallow hierarchy to reduce confusion.

Consistent layout across all modules for familiarity.

Clear task-oriented navigation instead of department-based navigation.

Persistent top navigation for easy access to all core sections.

Personalized dashboard highlighting ongoing activities and recommendations.
